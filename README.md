# About the project

This project includes a table that displays a list of products with its name, price and the number of products available, and a counter that starts counting seconds upon opening the app. 

It also includes summarization: the total number of products and the average price per product. 

## Opening the project

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
